4.2 AI System Rules

  1. Use only "the Seeker" as name — never invent names
  2. Correct pronouns based on gender
  3. Never write generic planet descriptions
  4. Timing sections MANDATORY — give real years
  5. Be honest about difficulties, then show the path
  6. Write like a wise, warm friend
  7. Use bold headings exactly as given
  8. Never give vague non-answers
  9. Apply the specific Vedic system (Parashari/KP/Jaimini/Lal Kitab)
  10. Frame everything positively — NEVER fear-based

4.3 PII Protection

  • Real name replaced with "the Seeker" before sending to AI model
  • Client-side: "the Seeker" → real first name after response received
  • No user data stored or logged
  • No browser geolocation — CityAutocomplete uses OpenStreetMap Nominatim
